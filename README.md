# libimgtype

![libimgtype](snap.jpg)

[imgtype](https://github.com/kjunichi/imgtype) is a like imgcat command for windows.

libimgtype is a library version of imgtype.

# How to use

you may need vcvarsall.bat, add gcc path.

```shell
makelib.bat

cd examples
make.bat

set path=%path%;..
main.exe
```
